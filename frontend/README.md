# HelpDesk Academy Frontend

React + Vite client for the LMS.

## Included Pages
- Landing page
- Login / Register
- Admin: dashboard, users, modules, trainees, analytics
- User: dashboard, modules, module viewer, quiz, progress, certificates

## Setup
1. Install dependencies
   - `npm install`
2. Create `.env`
   - `VITE_API_BASE_URL=http://localhost:4000`
3. Run app
   - `npm run dev`
